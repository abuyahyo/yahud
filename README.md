# موسوعة اليهود واليهودية والصهيونية

قارئ ويب لِـ«موسوعة اليهود واليهودية والصهيونية» للدكتور عبد الوهاب المسيري —
سبعة مجلدات، نحو ٨٤٨٦ صفحة — مع تصفّحٍ بالفصول وبحثٍ شامل، يعمل دون اتصال.

**المصدر:** [shamela.ws/book/2074](https://shamela.ws/book/2074)
**الموقع:** [abuyahyo.github.io/yahud](https://abuyahyo.github.io/yahud)

## المساعد الذكي (سؤال وجواب بالأوزبكية)

زرّ ✦ في الشريط العلويّ يفتح مساعدًا يجيب عن أسئلتك **بالأوزبكية** اعتمادًا على نصّ
الموسوعة نفسها: يستخرج كلمات بحثٍ عربية من سؤالك، يجد الصفحات الموافقة محلّيًّا،
ثمّ يصوغ إجابةً مع إحالاتٍ قابلة للنقر إلى الصفحات (ص N).

- يعمل في المتصفّح مباشرةً — لا خادم وسيط؛ يُحفَظ مفتاحك في `localStorage` فقط.
- مزوّدان قابلان للاختيار من ⚙:
  - **Google Gemini** — مجّانيّ (مفتاح من [aistudio.google.com](https://aistudio.google.com/apikey))، وهو الافتراضيّ.
  - **Claude** — مدفوع، إجابات أدقّ (مفتاح من [console.anthropic.com](https://console.anthropic.com/settings/keys)).

## البنية

- `index.html` — قارئ صفحةً فصفحة مع شريط أدوات (فهرس، بحث، المساعد الذكي، الإعدادات: حجم الخطّ والوضع الليلي).
- `meta.json` — معلومات المجلدات وحدودها.
- `vol-1.json … vol-7.json` + `intro.json` — نصوص كلّ مجلد (تُحمَّل عند الفتح فقط).
- `sw.js` — خدمة عامل (Service Worker) للعمل دون اتصال.
- `manifest.webmanifest` — يجعل القارئ قابلًا للتثبيت كتطبيقٍ على الشاشة الرئيسيّة.

## البناء من الصفر

```bash
py -3 fetch.py        # يجلب 8486 صفحة من shamela.ws إلى raw/
py -3 build_data.py   # يبني meta.json و vol-*.json
```
